CineWave is a **React TypeScript movie website** that uses the **TMDb API** to display movies, search, filter, and explore trending content. This project demonstrates a modern frontend application with React, TypeScript, and API integration.

---

## Features

- Browse popular and trending movies
- Search for movies by title
- Filter movies by genres
- View movie details (overview, release date, rating)
- Responsive design for desktop and mobile
- TypeScript for type safety
- Modular React components

---

## Tech Stack

- **Frontend:** React.js with TypeScript  
- **State Management:** React Hooks  
- **API:** TMDb API  
- **Styling:** CSS / Tailwind (if used)  
- **Version Control:** Git & GitHub
